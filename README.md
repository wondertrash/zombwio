# zombw.io
A primitive zombie survival game made for the Campfire Flagship game jam.

# About
Survive in a post-apocalyptic world with limited resources. Start with basic weapons and scavenge materials to craft better gear whilst managing hunger and fighting off zombies.

# Controls
- WASD - Move
- Mouse - Aim
- Left Click - Attack
- C - Crafting menu
- B - Building menu
- E - Interact with chests

## Gameplay Tips

- Eat berries to restore hunger and health
- Stay near campfires to heal and reduce hunger drain
- Copper is rare - use it wisely for turrets and crossbows
- Tank zombies (blue) require multiple hits but move slowly
- Fast zombies (red) are weak but aggressive
- Build the base of your dreams
